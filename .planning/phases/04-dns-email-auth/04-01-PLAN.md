  <action>
    Create the dns/dkim/ package with three components:

    **keygen.go** - DKIM 2048-bit RSA key generation:
    - `GenerateKeyPair(bits int) (*rsa.PrivateKey, error)` - generates RSA key pair (default 2048-bit per user decision). Use crypto/rsa.GenerateKey with crypto/rand.Reader.
    - `SaveKeyPair(privateKey *rsa.PrivateKey, dir string, selector string) error` - saves private key as PEM to `{dir}/{selector}.private.pem` and public key to `{dir}/{selector}.public.pem`. Use crypto/x509.MarshalPKCS1PrivateKey for private, x509.MarshalPKIXPublicKey for public. Set file permissions to 0600 for private key (matching Phase 1 pattern of 0600 for secrets).
    - `LoadPrivateKey(path string) (*rsa.PrivateKey, error)` - loads PEM-encoded private key from disk.
    - `PublicKeyBase64(publicKey *rsa.PublicKey) (string, error)` - returns base64-encoded DER public key (for DKIM TXT record content). Use x509.MarshalPKIXPublicKey then base64.StdEncoding.EncodeToString.

    **selector.go** - Time-based DKIM selector naming (per research pattern):
    - `GenerateSelector(prefix string, t time.Time) string` - format: `{prefix}-{YYYY}q{Q}` (e.g., "darkpipe-2026q1"). Quarter = (month-1)/3 + 1.
    - `GetCurrentSelector(prefix string) string` - returns selector for current quarter.
    - `GetNextSelector(prefix string) string` - returns selector for next quarter (for rotation planning).
    - `ShouldRotate(lastRotation time.Time) bool` - returns true if current quarter differs from lastRotation quarter.

    **signer.go** - DKIM message signing wrapping emersion/go-msgauth:
    - `type Signer struct` - holds domain, selector, private key.
    - `NewSigner(domain, selector string, privateKey *rsa.PrivateKey) *Signer`
    - `Sign(r io.Reader) ([]byte, error)` - signs an email message using dkim.Sign from emersion/go-msgauth with relaxed/relaxed canonicalization. Returns the signed message bytes. Sign headers: From, To, Subject, Date, Message-ID, MIME-Version, Content-Type.

    Tests should cover:
    - Key generation produces valid 2048-bit key
    - Key save/load roundtrip preserves key material
    - PublicKeyBase64 produces valid base64 that can be decoded back
    - Selector generation for different quarters (Q1-Q4)
    - ShouldRotate detects quarter boundaries
    - Signing produces message with DKIM-Signature header
    - Signed message verifiable with public key

    Add dependencies: `go get github.com/emersion/go-msgauth`
  </action>

    **records/spf.go** - SPF record generation:
    - `type SPFRecord struct { Domain, Value string }` - represents the TXT record.
    - `GenerateSPF(domain string, relayIP string, additionalIPs []string) SPFRecord` - generates `v=spf1 ip4:{relayIP} {additional} -all`. Use explicit ip4: mechanism (not include:) to minimize DNS lookup count per pitfall #1. Always use `-all` (hard fail) as default. Warn if additionalIPs would push past 10 DNS lookups.

    **records/dkim.go** - DKIM TXT record generation:
    - `type DKIMRecord struct { Domain, Selector, Value string }` - Domain is `{selector}._domainkey.{domain}`.
    - `GenerateDKIMRecord(domain string, selector string, publicKeyBase64 string) DKIMRecord` - generates value `v=DKIM1; k=rsa; p={publicKeyBase64}`. Ensure single-line output (no line breaks in TXT record value per pitfall #3).

    **records/dmarc.go** - DMARC record generation:
    - `type DMARCRecord struct { Domain, Value string }` - Domain is `_dmarc.{domain}`.
    - `type DMARCOptions struct { Policy, SubdomainPolicy, Percentage, RUA, RUF string }`
    - `GenerateDMARC(domain string, opts DMARCOptions) DMARCRecord` - generates `v=DMARC1; p={policy}; sp={subdomain_policy}; rua=mailto:{rua}; ruf=mailto:{ruf}; pct={pct}`. Include sp= tag for subdomain protection per pitfall #5. Default: p=none (monitoring first per pitfall #4), sp=quarantine, pct=100. Document recommended progression: none -> quarantine -> reject.

    **records/mx.go** - MX record generation:

    - `GenerateMX(domain string, relayHostname string, priority int) MXRecord` - generates MX record pointing to cloud relay FQDN. Default priority 10.
